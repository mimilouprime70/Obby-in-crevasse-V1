# Obby-in-crevasse-V1

Hello, here is my first Roblox game in its first official version. The goal is simple : make your way through the course by overcoming various obstacles without dying, all while staying in a crevice. 

If you want to play : https://www.roblox.com/fr/games/75201856168817/Obby-in-crevasse-v1

If you want to check the game's source files, you can open the “.rbxl” file (provided you have a Roblox Studio account and the application installed on your computer)
 There are also the game's source files, but it's more complicated to view the code. 

 Enjoy the game :)
